# index-ssl-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，主要用于测试、实验或静态展示目的。

## 项目简介

这是一个轻量级的静态页面仓库，集中存放独立的 HTML 文件。每个文件均为自包含的页面，可单独通过浏览器打开或部署到任意静态托管服务。仓库本身不提供任何动态服务、后端逻辑或外部依赖。

## 目录说明

```
index-ssl-lab/
├── .gitignore          # 忽略文件配置
├── README.md           # 本文件
├── index.html          # 仓库主页（可选）
└── pages/              # 存放独立 HTML 页面的目录
    ├── example1.html
    ├── example2.html
    └── ...
```

- `pages/` 目录为页面归档主目录，所有独立 HTML 文件均放置于此。
- 每个页面应遵循无外部资源依赖原则（或仅引用 CDN 公共库），以确保离线或简单部署可用。

## 页面归档说明

- 所有页面均为静态 HTML，不包含服务器端脚本。
- 文件名应简洁且具有描述性，避免使用中文字符或特殊符号。
- 页面内部可包含 CSS 和 JavaScript，但建议内联或引用稳定公共 CDN 资源。
- 本仓库不追踪页面访问量，不收集用户信息。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立页面，但请确保内容不违反 GitHub 社区准则。
- 提交前请检查页面是否能在无网络环境下正常加载（外部资源除外）。
- 本仓库仅作为归档用途，不保证所有页面长期可用或更新。
- 对于长期未维护的页面，可能会被清理或归档。
